## Core Principle

Pharmacologic therapy for T2D is no longer selected based on A1C reduction alone. The 2026 ADA algorithm prioritizes **comorbidity-driven selection**: the presence of cardiovascular disease, heart failure, CKD, obesity, or MASLD determines which agent class is preferred — independent of glycemic status.

## The Algorithm (Figure 9.4 Framework)

### Step 1: Assess for Compelling Comorbidities

| Comorbidity | Preferred Agent | Evidence |
|-------------|----------------|---------|
| **Established ASCVD or high CV risk** | GLP-1 RA with proven MACE benefit (semaglutide, liraglutide, dulaglutide) | LEADER, SUSTAIN-6, REWIND |
| **Heart failure (HFrEF or HFpEF)** | SGLT2i (empagliflozin or dapagliflozin) | EMPA-REG, EMPEROR-Preserved, DAPA-HF |
| **CKD (eGFR 20–60 or UACR >300)** | SGLT2i first (empagliflozin, dapagliflozin); GLP-1 RA if SGLT2i not tolerated | CREDENCE, DAPA-CKD, FLOW |
| **MASLD / MASH** | GLP-1 RA (semaglutide preferred) | Phase 3 NASH data; MASH liver outcomes |
| **Obesity / weight reduction priority** | GLP-1 RA or dual GIP/GLP-1 RA (tirzepatide) | SURMOUNT-1, STEP-1 |
| **Minimize hypoglycemia** | GLP-1 RA, SGLT2i, DPP-4i, metformin | No hypoglycemia in monotherapy |
| **Minimize cost** | Metformin, SU (generic), glargine biosimilar | UKPDS; generic availability |

### Step 2: If No Compelling Comorbidity — Personalize Preferences

After comorbidities are addressed, individualize based on:
- A1C above target → choose agent with A1C-lowering potency
- Patient preference (injection aversion → oral options)
- Cost and insurance

### Step 3: Consider Initial Combination Therapy

- New evidence supports starting with **two agents** at diagnosis if A1C is ≥2% above goal (>9%)
- Combination options: metformin + GLP-1 RA, or metformin + SGLT2i (especially with ASCVD/HF/CKD)
- Avoids delays from stepwise escalation in high-risk patients

## Agent Classes at a Glance

| Class | A1C Reduction | Weight | Hypo Risk | Cost |
|-------|--------------|--------|-----------|------|
| Metformin | 1–2% | Neutral/−1 kg | None | Very low |
| GLP-1 RA | 1–2.5% | −3–15% BW | None | High |
| SGLT2i | 0.5–1% | −2–3 kg | None | High |
| DPP-4i | 0.5–0.8% | Neutral | None | Moderate |
| Sulfonylurea | 1–2% | +2–3 kg | Moderate | Very low |
| TZD (pioglitazone) | 1–1.5% | +2–5 kg | None | Low |
| Insulin | 1.5–3.5% | +2–4 kg | High | Moderate |

## Deprescribing Guidance

Reduce or eliminate:
- Sulfonylureas when adding GLP-1 RA or insulin → hypoglycemia overlap
- DPP-4i when adding GLP-1 RA → no additive benefit (same mechanism pathway)
- Insulin dose reduction (often 20–50%) when adding GLP-1 RA
- Reduce or discontinue insulin if T2D enters remission (post-bariatric surgery, significant weight loss)
